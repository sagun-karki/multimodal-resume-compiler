# LATEX SANITIZATION & HEURISTIC CHECKS

## Purpose
Prevent compilation crashes and run quick syntactic sanity checks on the generated LaTeX strings before invoking the compiler process.

## Key Operations
1. **LaTeX Special Character Escaping:**
   Ensures standard text does not conflict with LaTeX parser control characters by escaping raw symbols:
   - `&` is escaped to `\&`
   - `%` is escaped to `\%`
   - `#` is escaped to `\#`
   - Unescaped underscores `_` are converted to `\_`.
2. **XeLaTeX Template Integrity Pre-Checks:**
   - Validates that matching markers and sections are preserved.
   - Ensures that generated structures fit the `awesome-cv.cls` class style formatting specifications without introducing illegal packages.
3. **Factuality Substring Verification:**
   - Performs a pre-compile lookup check to confirm that key technical credentials and projects generated by the model match exact substrings in the input `user_profile.md`, blocking synthetic fabrications.
